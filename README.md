# homebrew-exarp

Homebrew tap for [Exarp](https://github.com/flint-dominic/exarp) — behavioral backup intelligence.

## Install

```bash
brew tap flint-dominic/exarp
brew install exarp
```

## Usage

```bash
exarp scan /path/to/files
exarp check /path/to/files
exarp watch /path/to/files --interval 300
```

## What is Exarp?

One binary. Scans your files. Learns what's normal. Screams when it's not.

Exarp monitors file entropy to detect ransomware and other threats. No install, no account, no cloud. Delete the binary to uninstall.

[Full documentation →](https://github.com/flint-dominic/exarp#readme)
